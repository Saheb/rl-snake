# Snake Curiosity Exploration

An interactive exploration of "Curiosity-driven Exploration by Self-supervised Prediction" (Pathak et al., 2017).

Built with [marimo](https://marimo.io).

## Deployment Instructions
1. Create a new Space on Hugging Face.
2. Select **Docker** SDK.
3. Upload the files in this directory.
